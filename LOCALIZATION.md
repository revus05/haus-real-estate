# Локализация для Беларуси / Локализация для России

## 🇧🇾 Статус локализации: ✅ ПОЛНАЯ

Платформа Haus Real Estate полностью локализирована для белорусского/русского рынка.

### 📝 Что локализировано

#### Язык
- **Все UI текст на русском** ✅
  - Все метки форм
  - Все кнопки
  - Все сообщения об ошибках
  - Все заголовки и описания
  - Все уведомления (toast)

#### Валюта
- **Валюта по умолчанию: BYN** (Беларусский рубль)
- **Поддерживаемые валюты**: BYN, USD, EUR, UAH
- **Форматирование цен**: Локаль ru-RU (например: "1 500,00 BYN")

#### География
- **Город по умолчанию**: Минск (вместо New York)
- **Доступные районы Минска**:
  - Центральный
  - Советский
  - Октябрьский
  - Партизанский
  - Фрунзенский
  - Ленинский
- **Примеры адресов**: ул. Ленина, дом 1 (вместо 123 Main Street)

#### Телефон
- **Формат номера**: +375 29 XXX-XX-XX
- **Валидация**: Беларусский формат с кодами операторов (29, 33, 44, 25)
- **Плейсхолдеры**: Везде указан правильный формат +375 29 XXX-XX-XX

#### Дата и время
- **Локаль**: ru-RU (русская локаль)
- **Формат**: "13 апреля 2024 г." (длинный формат)

### 📁 Файлы с локализацией

#### Основные файлы:
- `messages/ru.json` - Все строки переводов
- `lib/translations.ts` - Утилита для переводов
- `i18n.config.ts` - Конфигурация i18n
- `lib/validations/` - Валидация на русском с беларусским форматом телефона

#### Обновлённые компоненты (45+ файлов):
- ✅ Компоненты аутентификации (login, register)
- ✅ Компоненты каталога (фильтры, сортировка, поиск, карточки)
- ✅ Компоненты деталей объекта (информация, форма заявки, карточка агента)
- ✅ Компоненты управления объявлениями (форма, загрузчик, статусы)
- ✅ Компоненты заявок (таблица, выбор статуса)
- ✅ Компоненты админки (управление пользователями, модерация)
- ✅ Компоненты макета (header, footer)

### 🔄 Примеры локализации

#### Форма регистрации
```
Полное имя → Полное имя
Электронная почта → Электронная почта
Пароль → Пароль
Телефон → Телефон (+375 29 XXX-XX-XX)
Создать аккаунт → Создать аккаунт
```

#### Фильтры каталога
```
Deal type → Тип сделки (Все, Продажа, Аренда)
Property type → Тип недвижимости (Квартира, Дом, Студия, и т.д.)
Price → Цена (диапазон BYN)
Rooms → Комнаты
Area → Площадь (м²)
City → Город (Минск)
```

#### Форма объявления
```
Price currency default → BYN вместо USD
Address placeholder → ул. Ленина, дом 1
District placeholder → Октябрьский
Validation errors → На русском
```

### 💰 Валюта

```typescript
// Поддерживаемые валюты:
BYN - Беларусский рубль (по умолчанию)
USD - Доллар США
EUR - Евро
UAH - Украинская гривня

// Форматирование:
Intl.NumberFormat("ru-RU", { 
  style: "currency", 
  currency: "BYN" 
})
// Результат: "150 000,00 BYN" или "150 000,00 р."
```

### 📞 Валидация телефона

```typescript
// Беларусский номер: +375 XX XXX-XX-XX
// Поддерживаемые коды:
29 - A1
33 - Velcom
44 - МТС
25 -life:)

// Примеры:
✅ +375 29 123-45-67
✅ (0)29 123-45-67
✅ +375 (29) 123-45-67
❌ +1 234 567 8900 (не беларусский)
```
